TÉCNICAS PARA ASEGURAR EL SISTEMA

Podemos enunciar varias como lo son:

  • Técnicas de Desarrollo: Las cuales deben cumplir con los criterios de seguridad al uso para todo el software.
  • Medidas de Seguridad Física: Sistemas anti incendios, vigilancia de los centros de procesos de datos, sistema de protección contra inundaciones, etc.
  • Codificar la Información: Se debe realizar una criptología, criptografía y criptociencia para todos los trayectos en donde circule información.
  • Contraseñas Difíciles de Averiguar.
  • Vigilancia de Red para Evitar Robo de Información.
  • Redes Perimétricas de Seguridad o DMZ: Generan reglas de acceso fuertes entre los usuarios y servidores no públicos y los equipos publicados.
  • Tecnologías Repelentes o Protectoras: Se pueden utilizar cortafuegos, sistemas de detección de intrusos, antivirus, llaves para protección de software, etc.
  • Mantener los sistemas de información con las actualizaciones que más impacten en la seguridad.
  • Copias de seguridad o sistemas de respaldo remoto.
  • Controlar el acceso a la información por medio de permisos centralizados y mantenidos.
